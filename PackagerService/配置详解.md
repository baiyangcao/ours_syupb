# 配置详解

## task节点
### 概述
定义导出任务，及导出任务的根目录

### 属性列表
| 属性名称 | 属性说明 |
|:------- | ------- |
| name | 任务名称，随意填写，不影响导出结果，用于日志中标识 |
| outputfolder | 输出目录属性，作为本次任务导出的根目录 |

## flow节点
### 概述
定义导出流程，并取出导出所用的数据源

### 属性列表
| 属性名称 | 属性说明 |
|:------- | ------- |
| name | 流程名称，随意填写，不影响导出结果，用于日志中标识 |
| sql | 获取数据所用的sql语句名称，用SQLGetter.GetSqlByAttr方法获取具体的SQL语句 |
